创建Registration Form

作业背景：

网页表单是任何交互性网站的核心部分，无论是注册、登录、提问还是购物，表单都扮演了关键角色。为了强化你对HTML和基础CSS的理解，我们为你设计了这个作业，让你从实践中学习。

作业要求：

表单结构：你的表单应该至少包含以下字段：

  1. First Name (text input field)
  2. Last Name (text input field)
  3. Email Address (appropriate input type)
  4. Password (password input field with appropriate attributes to ensure security)
  5. Confirm Password (password input field)
  6. Date of Birth (date picker)
  7. Gender (radio buttons: Male/Female/Other)
  8. About Me (textarea)
  9. Choose Your Profession (dropdown list with options like "Web Development", "Data Analytics", "UI/UX Design", etc.)

布局和样式：

  1. 使用CSS为你的表单元素添加基本样式，如边框、背景颜色、文本颜色等。
  2. 确保所有元素在各种设备和屏幕尺寸上都有清晰的布局和良好的可读性。
  3. 考虑元素间的间距，确保表单整体布局美观。

交互：

  1. 为提交按钮添加hover效果。
  2. 使用HTML5验证功能确保在提交之前填写了所有必需字段。
  3. 确保电子邮件和密码字段有适当的格式限制。

代码清晰性：确保你的代码结构清晰，HTML和CSS均应有合适的注释说明。


作业提交：

方式一：GitHub链接提交

  1. 将你的HTML和CSS 文件上传到 GitHub 仓库，确保项目结构清晰且完整；
  2. 上传可访问的仓库链接，点击“确认提交”，提交成功。

方式二：ZIP文件提交

  1. 将你的HTML和CSS代码保存在一个名为 "Registration_Form" 的文件夹中；
  2. 将文件夹压缩为 ZIP 文件；
  3. 上传zip文件，点击“确认提交”，提交成功。

请确保提交的代码结构完整、清晰，并能够正常运行。
